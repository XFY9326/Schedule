## 概述

此处将会介绍部分参数中的对象的数据结构与方法

## 登录页面信息 LoginPageInfo

在首次加载登录界面的同时，用于存储实际登录所需的信息  

``` kotlin
// 构造函数
// 参数：验证码Url，登录所需的Post参数（来自Ktor），其他需要传递的信息（放在Bundle内）
constructor(captchaUrl: String? = null, loginParams: Parameters? = null, bundle: Bundle? = null)

// 特殊方法
// 返回一个空的LoginPageInfo
val LoginPageInfo.Empty 
```

## 网页内容 WebPageContent

用于存储与传输网页内容  

``` kotlin
// 参数
// 当前HTML内容
val htmlContent: String

// <iframeContent>中内容
val iframeContent: Array<String>

// <frameContent>中内容
val frameContent: Array<String>

// WebCourseProvider中在调用onValidateCourseImportPage后，通过WebPageInfo的providedContent传入的自定义内容
// 用于在校验网页是否可以导入后，直接从WebCourseProvider中将需要解析的部分网页传给WebCourseParser
val providedContent: Array<String> = emptyArray()

// 获取第一个的providedContent，否则抛出CourseAdapterException.Error.PARSE_PAGE_ERROR的错误
// 若使用providedContent只传递一个数据，建议使用该方法获取数据
fun requireFirstProvidedContent(): String

// 获取拥有至少指定数量元素的providedContent数组，否则抛出CourseAdapterException.Error.PARSE_PAGE_ERROR的错误
// 若使用providedContent传递多个数据，建议使用该方法获取数据
fun requireProvidedContentWithMinimumSize(minSize: Int): Array<String>
```
